This project is a starting point for a Flutter application.

## App Functionalities

1. Valid Todo list

    a. There should be no “empty” item in the list

2. Unique Todo list

    a. There should be no duplicate item in the list.

    b. Adding a duplicate item will warn the user

3. Removable Todo list

    a. There should be a “remove” button next to the item

    b. The item should be removed from the list by clicking on the “remove” button.

4. Editable Todo list

    a. There should be an “edit” button next to each item.

    b. After clicking on the “edit” button, the input field will have the text of the item and users
    can edit the item by click ‘update’ button

    c. Use the same input widget as you add a new Todo item.

5. Filter Todo list

    a. When the user starts typing in the input widget, apply a simple text matching to the
    list of todo and only show the filtered result.

    b. When there is no matched result, please show “No result. Create a new one instead

6. Mark as Complete

    a. There should be an “Mark as Complete”/“Mark as Incomplete” button next to the item 

    b. Display the todo as strikethrough text if the todo is marked as completed.
